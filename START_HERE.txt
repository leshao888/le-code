--------------------------------------------------------------------------------
                            HOW TO START
--------------------------------------------------------------------------------

You have THREE options to start the program:


OPTION 1: Double-click the batch file (EASIEST)
------------------------------------------------
Simply double-click this file in File Explorer:
   > start_lecode.bat

This will open a new terminal window and run the program automatically.


OPTION 2: Using Command Prompt (CMD)
------------------------------------
1. Press Win + R on your keyboard
2. Type "cmd" and press Enter
3. Type the following commands:

   cd E:\AIProject\le-code\le-code
   python main.py


OPTION 3: Using PowerShell
--------------------------
1. Right-click the Start button
2. Select "Windows PowerShell"
3. Type the following commands:

   cd E:\AIProject\le-code\le-code
   python main.py
